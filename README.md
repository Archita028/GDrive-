Google Drive Clone (Spring Boot + React)

A full-stack Google Drive clone application that allows users to upload, store, view, and manage files locally.
The backend is powered by Spring Boot, and the frontend is built with React.js.
This project demonstrates secure file handling, RESTful APIs, and component-based UI architecture.
